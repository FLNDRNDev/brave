# brave
Brave: an Open Video Platform. Freedom to speak. Power to change.

## Getting Started

First, run the development server:

```bash
bun run dev
# or
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
